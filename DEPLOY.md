# CoopUI — Quick Install

1. Extract this zip into your **MacroQuest2 folder** (merge when prompted)
2. First time? Copy `config_templates/*` into `Macros/` (sell_config, shared_config, loot_config), or run ItemUI and use first-run default protection (Config window)
3. In-game: `/lua run itemui` and `/lua run scripttracker`
4. Toggle windows: `/itemui` and `/scripttracker`
5. Sell: `/dosell` — Loot: `/doloot`

**Updating?** Extract and overwrite. Your config INI files are preserved.

For detailed instructions see **[docs/INSTALL.md](docs/INSTALL.md)**.

## Requirements

- MacroQuest2 with Lua support (mq2lua) and ImGui
